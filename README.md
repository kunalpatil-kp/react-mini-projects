# React Mini Projects

A collection of **3 interactive React mini projects** built using fundamental React concepts. Perfect for learning and practicing React basics.

## Projects

### 1. 🎟️ Lottery Game
A fun lottery ticket generator where you win if the sum of 3 random digits equals **15**.

**Concepts Used:**
- **`useState`** Hook
- **Event Handling** (`onClick`)
- **Conditional Rendering** (`&&` operator)
- **State Management** for ticket and win condition
- **JavaScript Array methods** (`.join()`)

**Folder:** [`LotteryProject`](./LotteryProject)

---

### 2. ✅ Todo List App
A fully functional Todo List application with add, delete, and text transformation features.

**Concepts Used:**
- **`useState`** Hook (for todos and input)
- **Event Handling** (`onChange`, `onClick`, `onKeyPress`)
- **Conditional Rendering**
- **Array Methods** (`.map()`, `.filter()`, `.map()` for updating)
- **UUID** for unique task IDs
- **Form Handling**

**Folder:** [`TodoListProject`](./TodoListProject)

---

### 3. ✊ Rock Paper Scissors Game
Classic Rock-Paper-Scissors game against computer with score tracking and history.

**Concepts Used:**
- **`useState`** Hook (multiple states: choices, score, history)
- **Event Handling** (`onClick`)
- **Conditional Rendering**
- **Array Methods** (`.map()`, spread operator)
- **Game Logic** with conditional statements
- **History Tracking** using state

**Folder:** [`RockPaperScissors`](./RockPaperScissors)

---
